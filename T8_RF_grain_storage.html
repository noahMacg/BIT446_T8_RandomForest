Conclusion and Analysis

Key Random Forest Settings

max_sample --> This sets the sample size of each individual decision tree from which the RF model builds from. After iterating through values of the max_sample parameter of values from 200-800, we can see that there was no marked improvement in accuracy and precision or other notable metrics provided in the classification report. By using a max_sample of 200, and not changing any other parameters, we are still able to obtain the same level of predictions with lower max_sample values and subsequently have faster training and use less computational resources.

max_leaf_nodes --> This sets the max number of end classifications the tree can predict. Using a similar iteration as above, the max_leaf_node parameter was evaluated from 3-10 showing a plateau at 5. Changing the value from 3 to 5 increased all classification metrics except a slight decrease of the precision for the 1 target variable (bin purchased). Accuracy from 89% to 91%.

max_depth --> This sets the max depth the tree can grow to. I conducted a few iterations (not displayed) of adjusting the max_depth. I found that around a max_depth of 4 the model still performed identical to the classification report from the best model. A more thorough investigation would need to be conducted, but using 4 may help with reducing complexity and overfitting.

n_estimators --> This controls how many individual trees are the ensemble. I also tested different values for this and found that a much simpler ensemble created slightly better predictions; 91% accuracy to 92% accuracy. This may have also been creating a complicated forest and causing overfitting. By decreasing this value we can also significantly decrease computation.

max_features --> Adds randomness into feature selection, thereby increasing randomness into the ensemble of trees to create diversity. I briefly tried other values and it did not seem to have a meaningful effect.

KDS Metrics Report Plots
The metrics plots for lift, decile-wise, cumulative gain, and KS statistics showed promising results. The lift plots showed a range of ~ 2.7-3.6 times more predictive power (lift) than random guessing in the top 30% of customers ranked by probability. Cumulative gain showed that we can capture approximately 82% of all potential responders by targeting the top 30% of participants, and a KS score of 75.4 shows good/great separation of probabilities at decile 3. With this insight, it will be most beneficial to target marketing efforts to the top 2-3 deciles to gain the most profitable results.

Confusion Matrix Basic overview of the model using confusion matrix shows 164 correct predictions on storage bins not purchased and 54 correct predictions of storage bins purchased. There are low false negatives (5) and moderate false positives (17).
